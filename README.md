# Security Lab Algorithm Demonstrator

Interactive, client-side demos for classical and modern cryptography algorithms used in the Security Lab syllabus.

## Included demos

- Caesar Cipher
- Playfair Cipher
- Hill Cipher (2x2)
- Vigenere Cipher
- Rail Fence and Columnar Transposition
- DES (Toy Feistel Demo)
- RSA (small primes)
- Diffie-Hellman
- MD5
- SHA-1
- DSS (DSA demo)

## Run locally

1. Install dependencies:
	npm install
2. Start the dev server:
	npm run dev

## Notes

- This project is for learning and lab demonstrations only.
- Toy and small-parameter demos are not secure for real-world usage.
